## About Shopper

[Shopper](https://laravelshopper.dev) is an open source e-commerce administration built on top of Laravel for Laravel developers to create & manage online shop with multi-vendor.

## Packages

- [Shopper](https://github.com/shopperlabs/shopper) - A fully-featured Laravel admin panel.
- [Core](https://github.com/shopperlabs/core) - E-commerce domain logic (Models, Actions, Enums, Contracts).
- [Cart](https://github.com/shopperlabs/cart) - Cart management with pipelines, discounts, taxes and session handling.
- [Payment](https://github.com/shopperlabs/payment) - Payment processing and provider integrations.
- [Shipping](https://github.com/shopperlabs/shipping) - Shipping providers integration.
- [Sidebar](https://github.com/shopperlabs/sidebar) - A Laravel Sidebar builder.
- [Stripe](https://github.com/shopperlabs/stripe) - Stripe payment provider adapter.
